# status/ — Current Project State (pointer)

For **dashboard**, the single source of truth for "where is this project right now?" is [`src/_project-state/current-state.md`](../src/_project-state/current-state.md). `STATUS.md` here is a lightweight, high-level tracker kept for parity with the standard Vertex repo layout; it points at `current-state.md` rather than duplicating it.

## Maintenance discipline

- `src/_project-state/current-state.md` is updated to match reality at the end of every phase (per `AGENTS.md`).
- `STATUS.md` carries only the phase ledger (which phases are done) and links onward.
